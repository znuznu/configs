# Neovim configuration

Complete Neovim configuration based on [NvChad](https://github.com/NvChad/NvChad) suited for JavaScript/TypeScript, React, and Lua.

## Requirements

* [Neovim](https://neovim.io/) >= 0.11
* [Tree-sitter-cli](https://github.com/tree-sitter/tree-sitter/blob/master/crates/cli/README.md)
* [ripgrep](https://github.com/BurntSushi/ripgrep)
* a Nerd Font (optional)

## Installation

* Copy this folder in .config/nvim
* Run `nvim`
* Run `MasonInstallAll`
* Run `Lazy sync` and `checkhealth: lazy` to ensure everything is OK
